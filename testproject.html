Step by step how to use Git & Github

  1. Open terminal and cd into your desired directory. This is where you'll save your local files.

  2. In Github, create a new repository and paste first code block into terminal.

  3. After it's finished initializing, use command git add -A or git add (insert filename). Essentially, you are prepping your working stage before you commit the file.

  4. To check the status of your file, use command git status.

  5. At this point you, are ready to commit the file. To do this, use command git commit -m "Insert short description".

  6. Ok, you're not finished yet. You haven't actually loaded the file into Github. To do this, we'll be using the command git push origin master

  7. Voila! You're finished. Thumbs up, pat on your back, drink a swig of beer because you made it.
